SemanticApp is an interactive Knowledge Graph + Semantic Search web application built with Flask, spaCy, Sentence Transformers, and vis-network.

It allows you to:

📂 Upload CSV files containing triples (subject, relation, object)

✍️ Convert manual text into graph triples

🔎 Perform semantic search over graph nodes

📊 Monitor processing metrics in an Admin Dashboard

⭐ Collect user feedback ratings

📈 Track pipeline activity across the week

⚠️ This is an in-memory demo application (data resets when the server restarts).
